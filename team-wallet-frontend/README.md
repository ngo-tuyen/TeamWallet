# Team Wallet - フロントエンド

## セットアップ

### 前提条件
- Node.js 16以上
- npm または yarn

### インストール

1. **リポジトリをクローン**
```bash
git clone https://github.com/ngo-tuyen/TeamWallet/team-wallet-frontend.git
cd team-wallet-frontend
```

2. **依存パッケージをインストール**
```bash
npm install
```

3. **環境変数を設定**
```
VITE_API_URL=http://localhost:3000/api
VITE_APP_NAME=Team Wallet
```

4. **開発サーバーを起動**
```bash
npm run dev
```

アプリケーションは `http://localhost:5173` で実行されます
