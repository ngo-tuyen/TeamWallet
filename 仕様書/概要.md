# チーム管理システム仕様書

## 1. 概要

本システムは、ユーザーがログインし、複数のチームを作成・選択・管理できるWebアプリケーションです。ユーザーはチームに所属し、役割（admin / manager / user）に応じた操作が可能です。

---

## 2. 主な機能

| 機能名         | 説明                                                                 |
|----------------|----------------------------------------------------------------------|
| ユーザーログイン | メールアドレスとパスワードによる認証。JWTトークンを使用。               |
| チーム作成      | チーム名を入力して新規作成。作成者は自動的にadminとして登録される。       |
| チーム一覧表示   | ログインユーザーが所属するチームの一覧を取得・表示。                     |
| チーム選択      | 所属チームの中から1つを選択し、ダッシュボードへ遷移。                     |
| 取引（transactions) |チーム内で取引履歴記録・表示する                     |
| 画面遷移        | React Routerを使用して、ログイン → チーム選択 → ダッシュボードへ移動。     |

---

## 3. 画面構成

| 画面名         | URL              | 説明                                      |
|----------------|------------------|-------------------------------------------|
| ログイン画面     | `/`              | ユーザー認証を行う。                       |
| チーム選択画面   | `/select-team`   | 所属チームの一覧を表示し、選択可能。       |
| チーム作成画面   | `/create-team`   | 新しいチームを作成するフォーム。           |
| ダッシュボード   | `/dashboard`     | 選択されたチームの情報を表示（今後実装予定）。 |
| 取引画面         | `/transactions`  | チームの取引履歴を表示・登録する画面。     |
