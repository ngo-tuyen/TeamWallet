## 2. テーブル定義

### 2.1 users テーブル (ユーザー情報)

```sql
CREATE TABLE users (
    user_id INT PRIMARY KEY AUTO_INCREMENT,
    username VARCHAR(255) UNIQUE NOT NULL,
    password VARCHAR(255) NOT NULL,
    full_name VARCHAR(255) NOT NULL,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    
    INDEX idx_username (username)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COLLATE=utf8mb4_unicode_ci;
```

**カラム説明:**

| カラム | 型 | 制約 | 説明 |
|--------|-----|------|------|
| user_id | INT | PK, AI | ユーザーID |
| username | VARCHAR(255) | UQ, NN | ログイン用ユーザー名 |
| password | VARCHAR(255) | NN | bcryptハッシュ化されたパスワード |
| full_name | VARCHAR(255) | NN | ユーザーの表示名 |
| created_at | TIMESTAMP | DF | 作成日時 |

---

### 2.2 teams テーブル (チーム情報)

```sql
CREATE TABLE teams (
    team_id INT PRIMARY KEY AUTO_INCREMENT,
    team_name VARCHAR(255) NOT NULL,
    admin_id INT NOT NULL,
    description TEXT,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP,
    
    FOREIGN KEY (admin_id) REFERENCES users(user_id) ON DELETE CASCADE,
    INDEX idx_admin_id (admin_id),
    INDEX idx_created_at (created_at)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COLLATE=utf8mb4_unicode_ci;
```

**カラム説明:**

| カラム | 型 | 制約 | 説明 |
|--------|-----|------|------|
| team_id | INT | PK, AI | チームID |
| team_name | VARCHAR(255) | NN | チーム名 |
| admin_id | INT | FK, NN | チーム管理者 (user_id) |
| description | TEXT | NULL | チーム説明 |
| created_at | TIMESTAMP | DF | 作成日時 |
| updated_at | TIMESTAMP | DF | 更新日時 |

---

### 2.3 team_members テーブル (チームメンバー)

```sql
CREATE TABLE team_members (
    member_id INT PRIMARY KEY AUTO_INCREMENT,
    user_id INT NOT NULL,
    team_id INT NOT NULL,
    role ENUM('admin', 'manager', 'user') DEFAULT 'user',
    joined_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    
    FOREIGN KEY (user_id) REFERENCES users(user_id) ON DELETE CASCADE,
    FOREIGN KEY (team_id) REFERENCES teams(team_id) ON DELETE CASCADE,
    UNIQUE KEY unique_team_member (user_id, team_id),
    INDEX idx_team_id (team_id),
    INDEX idx_user_id (user_id)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COLLATE=utf8mb4_unicode_ci;
```

**カラム説明:**

| カラム | 型 | 制約 | 説明 |
|--------|-----|------|------|
| member_id | INT | PK, AI | メンバーID |
| user_id | INT | FK, NN | ユーザーID |
| team_id | INT | FK, NN | チームID |
| role | ENUM | DF | ロール (admin, manager, user) |
| joined_at | TIMESTAMP | DF | 参加日時 |

### 2.4 invitations テーブル (チーム招待)

```sql
CREATE TABLE invitations (
    invitation_id INT PRIMARY KEY AUTO_INCREMENT,
    team_id INT NOT NULL,
    user_id INT NOT NULL,
    invited_by INT NOT NULL,
    role ENUM('admin', 'manager', 'user') DEFAULT 'user',
    status ENUM('pending', 'accepted', 'rejected') DEFAULT 'pending',
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP,
    
    FOREIGN KEY (team_id) REFERENCES teams(team_id) ON DELETE CASCADE,
    FOREIGN KEY (user_id) REFERENCES users(user_id) ON DELETE CASCADE,
    FOREIGN KEY (invited_by) REFERENCES users(user_id) ON DELETE CASCADE,
    UNIQUE KEY unique_pending_invitation (team_id, user_id),
    INDEX idx_user_id (user_id),
    INDEX idx_team_id (team_id),
    INDEX idx_status (status)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COLLATE=utf8mb4_unicode_ci;
```

**カラム説明:**

| カラム | 型 | 制約 | 説明 |
|--------|-----|------|------|
| invitation_id | INT | PK, AI | 招待ID |
| team_id | INT | FK, NN | チームID |
| user_id | INT | FK, NN | 招待対象ユーザーID |
| invited_by | INT | FK, NN | 招待したユーザーID |
| role | ENUM | DF | 招待するロール |
| status | ENUM | DF | 招待状態 (pending) |
| created_at | TIMESTAMP | DF | 作成日時 |
| updated_at | TIMESTAMP | DF | 更新日時 |

### 2.5 categories テーブル (支出カテゴリ)

```sql
CREATE TABLE categories (
    category_id INT PRIMARY KEY AUTO_INCREMENT,
    team_id INT NOT NULL,
    category_name VARCHAR(255) NOT NULL,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    
    FOREIGN KEY (team_id) REFERENCES teams(team_id) ON DELETE CASCADE,
    UNIQUE KEY unique_category (team_id, category_name),
    INDEX idx_team_id (team_id)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COLLATE=utf8mb4_unicode_ci;
```

**カラム説明:**

| カラム | 型 | 制約 | 説明 |
|--------|-----|------|------|
| category_id | INT | PK, AI | カテゴリID |
| team_id | INT | FK, NN | チームID |
| category_name | VARCHAR(255) | NN | カテゴリ名 |
| created_at | TIMESTAMP | DF | 作成日時 |

### 2.6 incomes テーブル (収入情報)

```sql
CREATE TABLE incomes (
    income_id INT PRIMARY KEY AUTO_INCREMENT,
    team_id INT NOT NULL,
    amount DECIMAL(10, 2) NOT NULL,
    income_date DATE NOT NULL,
    memo VARCHAR(255),
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP,
    
    FOREIGN KEY (team_id) REFERENCES teams(team_id) ON DELETE CASCADE,
    INDEX idx_team_id (team_id),
    INDEX idx_income_date (income_date),
    INDEX idx_created_at (created_at)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COLLATE=utf8mb4_unicode_ci;
```

**カラム説明:**

| カラム | 型 | 制約 | 説明 |
|--------|-----|------|------|
| income_id | INT | PK, AI | 収入ID |
| team_id | INT | FK, NN | チームID |
| amount | DECIMAL(10, 2) | NN | 収入金額 (1円以上) |
| income_date | DATE | NN | 収入日 |
| memo | VARCHAR(255) | NULL | メモ |
| created_at | TIMESTAMP | DF | 作成日時 |
| updated_at | TIMESTAMP | DF | 更新日時 |

### 2.7 expenses テーブル (支出情報)

```sql
CREATE TABLE expenses (
    expense_id INT PRIMARY KEY AUTO_INCREMENT,
    team_id INT NOT NULL,
    category_id INT NOT NULL,
    amount DECIMAL(10, 2) NOT NULL,
    expense_date DATE NOT NULL,
    memo VARCHAR(255),
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP,
    
    FOREIGN KEY (team_id) REFERENCES teams(team_id) ON DELETE CASCADE,
    FOREIGN KEY (category_id) REFERENCES categories(category_id) ON DELETE CASCADE,
    INDEX idx_team_id (team_id),
    INDEX idx_category_id (category_id),
    INDEX idx_expense_date (expense_date),
    INDEX idx_created_at (created_at)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COLLATE=utf8mb4_unicode_ci;
```

**カラム説明:**

| カラム | 型 | 制約 | 説明 |
|--------|-----|------|------|
| expense_id | INT | PK, AI | 支出ID |
| team_id | INT | FK, NN | チームID |
| category_id | INT | FK, NN | カテゴリID |
| amount | DECIMAL(10, 2) | NN | 支出金額 (1円以上) |
| expense_date | DATE | NN | 支出日 |
| memo | VARCHAR(255) | NULL | メモ |
| created_at | TIMESTAMP | DF | 作成日時 |
| updated_at | TIMESTAMP | DF | 更新日時 |
